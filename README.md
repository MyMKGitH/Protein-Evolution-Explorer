Protein Evolution Explorer

Overview

Protein Evolution Explorer is an interactive bioinformatics application developed in Python using Streamlit and Biopython.

The platform enables users to retrieve protein sequences from UniProtKB, perform comparative sequence analysis, explore protein structures from the Protein Data Bank (PDB), and calculate structural similarity metrics.

Features

Sequence Analysis

- UniProtKB REST API integration
- Protein and gene search
- Cross-species homology exploration
- Pairwise sequence alignment
- Sequence identity calculation
- Comparative sequence visualization

Structural Bioinformatics

- PDB structure retrieval
- Interactive 3D molecular visualization
- RMSD calculation
- Kabsch structural superposition algorithm

Workspace Tools

- Persistent sequence basket
- Multi-protein comparison workspace
- Interactive Streamlit interface

Technologies

- Python
- Streamlit
- Biopython
- NumPy
- REST APIs
- Structural Bioinformatics
